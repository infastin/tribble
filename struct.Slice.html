Description [src] 

struct TrbSlice {
  void* (* at) (
    const TrbSlice* self,
    usize index
  );
  void* data;
  usize elemsize;
  usize start;
  usize end;
}

It is a data structure that is used to represent a portion of the data in any container whose elements can be directly accessed.

This example show how to sort an array:

i32 array[14] = { 27, 54, 14, 14, 48, 17, 44, 76, 95, 10, 97, 62, 55, 1 };

TrbSlice slice;
trb_slice_init(&slice, array, 4, 0, 14);

trb_inssort(&slice, (TrbCmpFunc) trb_i32cmp);

for (usize i = 0; i < 14; ++i) {
    printf("%d ", array[i]);
}

You should get such output:

1 10 14 14 17 27 44 48 54 55 62 76 95 97
