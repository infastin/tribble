Xoshiro128+ - is the best and fastest 32-bit generator from Xorshift family for 32-bit floating-point numbers. It passes all tests we are aware of except for linearity tests, as the lowest four bits have low linear complexity, so if low linear complexity is not considered an issue (as it is usually the case) it can be used to generate 32-bit outputs, too.
