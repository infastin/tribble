Struct

TribbleHashTable

Description [src] 

struct TrbHashTable {
  usize slots;
  usize used;
  usize keysize;
  usize valuesize;
  usize seed;
  TrbHashFunc hash_func;
  void* data;
  bool with_data;
}

A hash table with quadratic probing and size 2^n.

Structure members
slots

The number of buckets.
used

The number of used buckets.
keysize

The key size.
valuesize

The value size.
seed

The seed for the hash_func.
hash_func

The function for hashing keys.
data

User data.
with_data

Indicates whether TrbHashTable has been initialized with data or not.

Instance methods

trb_hash_table_add

Adds a new entry to the hash table.

trb_hash_table_destroy

Frees the hash table buckets.

trb_hash_table_free

Frees the hash table completely.

trb_hash_table_init

Creates a new TrbHashTable.

trb_hash_table_init_data

Creates a new TrbHashTable with the comparison function that accepts user data.

trb_hash_table_insert

Inserts an entry to the hash table. If the entry exists in the table, then replaces its value with the given one.

trb_hash_table_lookup

Searches for the entry in the hash table.

trb_hash_table_remove

Removes the entry from the hash table.

trb_hash_table_remove_all

Removes all entries from the hash table.
