Description [src]

Searches for the entry in the array using binary search and user data.

The array should be sorted if you want to use this function.

Parameters

array void*
 

The array to be searchen.
 The argument can be NULL.
 The data is owned by the caller of the function.
target void*
 

The pointer to the data to be found.
 The argument can be NULL.
 The data is owned by the caller of the function.
len usize
 

The length of the array.
elemsize usize
 

The size of each element in the array.
cmpd_func TrbCmpDataFunc
 

The function for comparing elements.
data void*
 

User data.
 The argument can be NULL.
 The data is owned by the caller of the function.
index usize
 

The pointer to retrieve the index of found value.
 The argument will be set by the function.
 The argument can be NULL.
 The called function takes ownership of the data, and is responsible for freeing it.

Return value

Returns: bool
 

TRUE if found, FALSE if not.
