Description [src]

It is analogous to vsnprintf(), except it allocates a string large enough to hold the output including terminating null byte.

This function is not directly available to language bindings

Parameters

buf char**
 

The buffer where the formatted string is to be placed.
 The data is owned by the caller of the function.
 The value is a NUL terminated UTF-8 string.
fmt const char*
 

The format of the string.
 The data is owned by the caller of the function.
 The value is a NUL terminated UTF-8 string.

The argument list.
