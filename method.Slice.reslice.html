Description [src]

Creates a new TrbSlice of another TrbSlice. If allocated on the heap, use free() to release the allocated memory.

This method is not directly available to language bindings.

Parameters

dst TrbSlice
 

The destination slice.
 The argument can be NULL.
 The data is owned by the caller of the function.
start usize
 

The start position in the dst data. It is relative to the src start position.
end usize
 

The end position in the dst data. It is relative to the src start position.

Return value

Returns: TrbSlice
 

A new TrbSlice. Can return NULL if an error occurs.
 The data is owned by the instance.
 The return value can be NULL.
