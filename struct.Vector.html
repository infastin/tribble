Struct

TribbleVector

Description [src] 

struct TrbVector {
  void* data;
  usize len;
  usize capacity;
  usize elemsize;
  bool clear;
}

A dynamic size array.

Structure members
data

The vector buffer.
len

The size of the vector.
capacity

The capacity of the vector buffer.
elemsize

The size of each element.
clear

Indicates whether elements should be cleared to 0 when allocated or not.

Instance methods

trb_vector_copy

Creates a copy of the vector.

trb_vector_destroy

Frees the vector buffer.

trb_vector_free

Frees the vector completely.

trb_vector_init

Creates a new dynamic size array.

trb_vector_insert

Inserts the element into the vector at the given index. If data is NULL, inserts a zero into the vector.

trb_vector_insert_many

Inserts elements into the vector at the given index. If data is NULL, inserts zeroes into the vector.

trb_vector_pop_back

Removes the last element from the vector.

trb_vector_pop_back_many

Removes last len elements from the vector.

trb_vector_pop_front

Removes the first element from the vector.

trb_vector_pop_front_many

Removes first len elements from the vector.

trb_vector_push_back

Adds the element to the end of the vector. If data is NULL, adds a zero to the end of the vector.

trb_vector_push_back_many

Adds elements to the end of the vector. If data is NULL, adds zeroes to the end of the vector.

trb_vector_push_front

Adds the element to the beginning of the vector. If data is NULL, adds a zero to the beginning of the vector.

trb_vector_push_front_many

Adds elements to the beginning of the vector. If data is NULL, adds zeroes to the beginning of the vector.

trb_vector_remove

Removes the entry at the given index from the vector.

trb_vector_remove_all

Removes all elements from the vector.

trb_vector_remove_range

Removes the range of elements from the vector.

trb_vector_require

Reserves memory in the vector buffer.

trb_vector_search

Searches for the entry in the vector.

trb_vector_search_data

Searches for the entry in the vector using user data.

trb_vector_shrink

Frees unused memory of the vector buffer.

trb_vector_slice

Slices the TrbVector.

trb_vector_steal

Steals the vector buffer. Array creates a new buffer.

trb_vector_steal0

Steals the vector buffer. Array’s buffer becomes NULL.
